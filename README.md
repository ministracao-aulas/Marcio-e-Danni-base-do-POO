### Alguns comandos GIT

```sh
# criando branch e mudando para ela
git checkout -b nome-da-branch

# mudando de branch
git checkout nome-da-branch

# adicionando arquivos alterados (todos)
git add .

# commit
git commit

# commit inline
git commit -m "xyz"

# git push [REMOTO] [NOME-DA-BRANCH]
git push origin nome-da-branch
```
